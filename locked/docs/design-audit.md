Note on overlaps: a handful of systemic defects were independently confirmed by two auditors and appear in two category tables — notably the L1168 blanket transition rule (Motion + Gestures), the L1163 global button curve (Motion + Gestures), the tutorial swipe (Motion + Gestures), confetti colors (Color + Brand), exclamation-mark copy (Components + Brand), the Fuel metric wall (Layout + Brand), and violet-accent scatter (Color + Brand). Counting each defect once, the deduplicated total is ~120 distinct findings.

## Ranked fix order

1. **Token foundation (Wave 1 prerequisite).** Extend the L1115 token layer with semantic status/accent/category/spacing/radius/type/motion tokens; migrate the 589 hex literals; delete the dead CSS_THEME block and the brittle `[style*=…]` patch selectors this unblocks (AUD-028..047 color-system rows, AUD-006/007 spacing+radius).
2. **Contrast Criticals.** CTA-gradient label contrast (2.80:1 worst point, 74 sites), input placeholder 2.55:1, light-theme accent-as-text 2.51:1, hardcoded status palette failing on light themes (AUD-028..031).
3. **Tap-target Criticals.** Dead 44pt CSS net (AUD-001), the 37 fixed-size + ~89 chip sub-44pt targets (AUD-002), WorkoutLog set-grid rebuild (AUD-003).
4. **Typography system.** Font consolidation decision (Barlow/DM Sans → system), rem migration + 17pt body floor + weight rebase (AUD-014..027).
5. **Motion foundation.** Kill the L1163/L1168 global curve rules (three Criticals resolve at once), ship the rAF spring utility + `usePressable`, interim curve swaps on all touchables, reduce-motion coverage for JS timers (AUD-059..077).
6. **Drag & scroll physics (Phase 4).** Reorder rebuild (one-gesture lift, 1:1 tracking, FLIP sibling springs, projection release, boundary haptics, touchcancel), mic FAB projection snap + mid-flight regrab, swipe-to-delete projection + rubber band, sheet pull-to-dismiss (AUD-078..097).
7. **Materials cleanup.** `prefers-reduced-transparency` fallback, un-stack glass (toast + scrims), unify scrim recipe, bar finish, kill dead selectors (AUD-048..058).
8. **Components & copy.** alert()→toast, error copy + retry, empty-state actions, sentence-case/vocabulary pass, nav remap gaps (AUD-098..114).
9. **Brand.** Gaming toggle + streak/badge gating, violet/blue/amber accent consolidation to orange, emoji reduction, Fuel/Profile three-question restructures (AUD-115..134).
