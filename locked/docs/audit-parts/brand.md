**Gaming-layer verdict:** No toggle exists anywhere (`lk_` key census confirms) — streak renders in the default Home header (L5498) and the badge/achievement grid in default Profile (L6589–6654), so the gaming layer bleeds into default views by construction; XP/levels/ranks/clans are not implemented at all (only streaks + badges).
**Accent census by role:** orange correctly owns most CTAs (~65 signature gradients of 114 total), but violet #8B5CF6 (×78) carries action/identity on Progress, Check-In, Settings, physique-AI and coach-goal CTAs; blue #3B82F6 (×70) carries water/BF/coach-food actions + protein; amber #F59E0B (×59) carries refeed CTA, warm-ups, nag banners + carbs; pink #EC4899 = fat — the P/C/F macro trio is the only near-defensible semantic use, everything else is scatter.
**Three-question test:** Pass — Home, Train, WorkoutLog, Cardio, Review, Coach, Progress, Shopping, Settings; Fail — Fuel (no dominant next action in a metric wall) and Profile (no next action; badges/settings compete).
