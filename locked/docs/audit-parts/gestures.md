**Summary of current mechanics:**
- **SplitBuilder reorder (L2672–2749):** 350ms touch-only long-press swaps the row for a dashed ghost and tracks a fixed full-width proxy pill by Y (eased 0.24s by blanket CSS L1168, so not 1:1); siblings never move — an insert spacer mounts/unmounts instantly; release splices the array with no spring, velocity, or projection.
- **WorkoutLog reorder (L3322–3447):** long-press only arms wiggle mode (the held finger is then dead — a second touch on a compact card starts the real drag, same proxy-pill/ghost/spacer mechanics as SplitBuilder), with instant-splice drop and 16ms interval edge auto-scroll.
- **Mic FAB (L13694–13844):** genuine 1:1 translate3d drag with a 6px tap threshold, but release snaps to the position-nearest corner via a fixed 0.35s ease curve with no velocity projection, and re-grabbing mid-snap teleports because state already sits at the target corner.
- **Swipe-to-delete (L3288–3305, L3699–3701):** tracks the finger 1:1 leftward only, then deletes instantly on a fixed −80px positional threshold with no projection, collapse animation, rubber band, or touchcancel handling.
