**Summary**
- Systemic counts: ~900 off-grid spacing values (top offenders: 6 ×259, 14 ×171, 11 ×80, 5 ×77, 7 ×70, 13 ×69, 9 ×55); 19 distinct border radii with ~290 odd-value uses and zero concentric nesting; 37 fixed-size sub-44pt buttons + ~89 small-padding chips + 3 undersized switches, all unprotected because the L1212–1213 min-44px CSS net can never match serialized style attributes.
- Screens audited: Home (+Weekly Recap), WorkoutLog (set grid, rest timer, tools, NumPad, drag/wiggle), Fuel header/tabs, Coach (tabs/instructions), Progress, Settings (toggles), Nav bar, app shell (floating bar, toasts, resume dialog) — Nav itself passes 44pt.
- Not audited in depth: Profile, Onboarding, TrainHub/SplitBuilder interiors, Shopping/Budget/Pantry, CardioLog, Review wizard, BetaAdmin, and the vanilla auth/paywall shell (L66–1057) — expect the same systemic spacing/target patterns there.
