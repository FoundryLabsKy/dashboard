**Summary.** Hex census (589 six-digit occurrences on 412 lines): #8B5CF6 ×66 (violet — partials/drop-sets, AI physique, entire Progress section, shoulders, settings headers), #EA580C ×65 (CTA gradient end-stop), #3B82F6 ×65 (protein macro, unilateral reps, body-fat, back muscle, info — token exists but unused), #F59E0B ×48 (warm-up sets, carbs, quads, PR chips, warnings), #EF4444 ×38 (error/delete), #22C55E ×34 (success), #F97316 ×33 (raw accent twin for alpha-concat), #10B981 ×19 (biceps, Smart Nutrition), #F5F5F5 ×17 (legacy text gray), #EC4899 ×15 (fat macro, glutes).
Measured core ratios (dark theme): #F5F5F7 text = 19.29:1 on #000 / 15.63:1 on #1C1C1E card; #A1A1AA muted = 8.19:1 / 6.64:1 — the neutral text system is excellent; slate/navy/midnight muted-on-card all pass (4.95 / 5.01 / 6.61); light muted #6E6E73 = 5.07:1 on white.
The failures are all chromatic: white-on-CTA-gradient 2.80:1 worst-point (74 instances), placeholder 2.55:1, orange-as-text 2.51:1 in light mode, and the entire hardcoded status/macro palette collapsing to 2.15–3.76:1 on light backgrounds because one fixed hex serves all five themes — token-layer extension is the prerequisite fix for two-thirds of the table.
